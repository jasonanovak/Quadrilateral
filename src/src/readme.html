Welcome to Quadrilateral!


What is Quadrilateral

Quadrilateral is a keyboard based window organizer for ChromeOS.

You can use Quadrilateral to assign a keyboard shortcut to resize a ChromeOS window to 1/6, 1/4, 1/3, or 1/2 of the screen and move the resized window to a fixed position (e.g. the top left 1/6th or right 1/3rd of the screen).

How to Use Quadrilateral

You need to go Chrome Settings > Extensions > Keyboard shortcuts to configure keyboard shortcuts. There are three configured by default:

  • Ctrl+Shift+4 - Resizes and moves the window to the left third of the screen.
  • Ctrl+Shift+5 - Resizes and moves the window to the center third of the screen.
  • Ctrl+Shift+6 - Resizes and moves the window to the right third of the screen.

After you configure Quadrilateral's shortcuts, you can then use those shortcuts to quickly resize and move the active ChromeOS window.

FAQs

Why aren't more shortcuts pre-configured?

The Chrome commands API only allows for four shortcuts to be configured by default.

I submitted a feature request to increase the number that can be configured by default. Feel free to +1 the bug (using the button in the upper right of that page) if you want to see this default changed.

How do does the author of Quadrilateral have it configured?

Here's the author's configuration:
  • Thirds - Left: Ctrl + Shift + 4
  • Thirds - Center: Ctrl + Shift + 5
  • Thirds - Right: Ctrl + Shift + 6
  • Sixths - Top Left: Ctrl + Shift + 1
  • Sixths - Top Center: Ctrl + Shift + 2
  • Sixths - Top Right: Ctrl + Shift + 3
  • Sixths - Bottom Left: Ctrl + Shift + 7
  • Sixths - Bottom Center: Ctrl + Shift + 8
  • Sixths - Bottom Right: Ctrl + Shift + 9
  • Quarters - Top Right: Alt + Home
  • Quarters - Top Left: Alt + Page Up
  • Quarters - Bottom Left: Alt + Page Down
  • Quarters - Bottom Right: Alt + End
  • Halves - Top: Alt + Shift + R
  • Halves - Bottom: Alt + Shift + V
  • Halves - Left: Alt + Shift + D
  • Halves - Right: Alt + Shift + G
The author chose these shortcuts as they didn't collide with anything in ChromeOS. Some of these are not ergonomic -- they were designed for use with a programmable keyboard (e.g. a ZSA Moonlander) where keys can be programmed with macros to send these shortcuts when those keys are held.

Why does Chrome say to proceed with caution and that this extension is not trusted by Enhanced Safe Browsing?

Quadrilateral is the author's first extension and, per Google, [for] new developers, it generally takes a few months to become trusted.

Why doesn't this work for windows from Android or Linux apps on ChromeOS?

Quadrilateral is a Chrome extension so it can only manage windows created by Chrome.

Why doesn't this work in Incognito mode?

It does! It just needs to be enabled in Extensions > My Extensions > Quadrilateral where "Allow in Incognito" needs to be flipped on.

Why does Quadrilateral have Halves - Left and Halves - Right when ChromeOS supports these window resize actions natively?

Because ChromeOS has an animation associated with it that is slower than Quadrilateral's resize and move.

I am using macOS or Windows or Linux, can I use this extension?

I would advise against it. It's meant for ChromeOS and other OSes and their window management systems haven't been tested with this extension.

If you're on Linux, your desktop environment probably has some equivalent functionality or add-ons (e.g. UltrawideWindows for KDE).

If you're on macOS, Rectangle.app provides similar functionality.

> I'm not familiar enough with Windows to say what the equivalent would be.
